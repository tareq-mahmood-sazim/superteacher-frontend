# Installation
```bash
yarn install
```

# Run
## RUN dev server
```bash
yarn start:dev
```

## BUILD compiled
```bash
yarn build
```

## START on production server
```bash
yarn start
```

# Commit Convention
``subject(ticket-code): message``

Valid subjects:
``build, chore, ci, docs, feat, fix, perf, refactor, revert, style, test``
